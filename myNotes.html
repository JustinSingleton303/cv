HTML Section

HTML is a mark up language used to give web pages their structure. Most webpages consist of more than just HTML these days, but HTML still has a very prominant role in the building of webpages online. Right now this page consists of just HTML.

Tags are a very important part of the HTML language. They help to define what is happening with the layout of the page, and can add stylistic features to the text. For instance, the "u" tag underlines all the text within it.

HTML
This tag declares that the contents within is HTML code.
h1 through h6
This tag makes a header. Headers have bold text and the lower numbers "h1", for example, have larger sized text than the larger numbers, h6 having the smallest text.
a
This denotes an anchor tag. These tags are good for placing links to other pages on the site and on the wider internet.
img
This tag is used to place images on the website.

Element An element is an individual component of an HTML document.

Attribute An attribute is a characteristic of an HTML element that can be set or changed. Font size or color are examples of attributes in HTML.

Style Style is an attribute within many elements used to set the visual aspects of the text within the tag having the style element within it.
